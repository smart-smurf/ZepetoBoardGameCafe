## Goal
 제페토 멀티플레이 서버로 통신 구조가 다소 복잡한 게임을 구현해보는것이 목표   
 턴제로 진행되는 보드게임만 서포트 (애초에 보드게임인데 턴제가 아닌게 있나..? 보드게임 안해서 잘 모르겠당!)
 
 구현 하려고 하는 게임은 아래와 같다. (구조상 동적으로 더 많은 보드게임을 추가 가능)
 
 - 틱택토       (2명까지 참여가능) 
 - 블랙잭       (4명까지 참여가능)
